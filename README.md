# Student Database

Full-stack Django app for managing student records with live class metrics.

## Features
- Add students (name, grade)
- Real-time stats: count, avg grade, min/max
- Responsive CSS grid layout
- Form validation + error handling
- Custom black theme

## Tech Stack
- **Backend:** Django 5.x, Python
- **Frontend:** HTML5, CSS Grid  
- **Database:** SQLite3

## Quick Start
```bash
python manage.py runserver
